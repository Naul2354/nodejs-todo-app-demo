# Todo list app
<br>

## cloning project

```
git clone https://github.com/Naul2354/nodejs-todo-app-demo.git
```

## install dependencies before run app

```
yarn install
```

<p>or</p>

```
npm install
```

## run server

```
yarn run dev
```

<p>or</p>

```
npm run dev
```

## open this app at your browser

```
http://localhost:8089
```
